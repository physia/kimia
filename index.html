Colors

Physia framework given easy color system integrated with material design, but you can customized fo fit your design.

usage

it was easy to use as you see!

Background color:

<div class="c-orng">
  Background color orange
</div>
Background color orange

Text color:

<div class="c-orng-t">
  Text color orange
</div>
Text color orange

Border color:

<div class="c-rd-b">
  Border color orange
</div>
Border color orange
palette

The palette give you 6 types of colors

type Pattern values
palette 
.c-*
 
values {rd,grn,bl,bl-gry,lt-grn,pnk,prpl,dp-prpl,indg,lt-bl,cn,tl,lm,ylw,ambr,orng,dp-orng,brwn,gry}
primary & secondary colors 
.c-{s,p}-*, .c-{s,p}
 
numbers {1 to 9} or {a1,a2,a4,a7} 
ex: c-s-5-t,c-s-a1
theme colors 
.c-*
 
numbers {0 to 20}
20 is #fff and 0 is #000
dynamic colors 
.c*, l-*, d-*
 
numbers {0 to 20}
20 is #fff and 0 is #000 on light mode
20 is #000 and 0 is #fff on dark mode
For more customization:
  l-*: only on light
  d-*: only on dark
static colors 
.c-*
 
values {err,wrn,scc}
more 
.c-*
 
values {i,n}
  n: transparent
  i: inherit
  c: currentColor
note: "c-*-f", fill in svg only in this 3 values
